## Результат выполнения программы

```
╔══════════════════════════════════════════════╗
║   ВАРИАНТ 3   |   РЕШЕНИЕ УРАВНЕНИЯ: 6/X=2   ║
╚══════════════════════════════════════════════╝

Нормализованное уравнение: 6/X=2
Левая часть:  6/X
Правая часть: 2

Обнаруженный оператор: '/' (деление)

Операнд 1: 6
Операнд 2: X

Расположение переменной X: ПРАВЫЙ операнд

Известный операнд: 6
Правая часть (b):  2

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ОТВЕТ: X = 3
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Проверка: 6 / 3 = 2 ✓ ВЕРНО
```

---

**Что делает программа (7 шагов):**

1. **Нормализация** — убирает пробелы, приводит к верхнему регистру
2. **Разбиение по `=`** — левая часть `6/X`, правая часть `2`
3. **Определение оператора** — перебирает `+`, `-`, `*`, `/` и находит `/`
4. **Разбиение на операнды** — `operand1 = "6"`, `operand2 = "X"`
5. **Расположение X** — X в правом операнде → формула `a / X = b → X = a / b`
6. **Вычисление** — `X = 6 / 2 = 3`
7. **Проверка** — подставляет X обратно и проверяет равенство

Программа **универсальна**: работает для всех четырёх операторов (`+`, `-`, `*`, `/`) и для X как в левом, так и в правом операнде.
